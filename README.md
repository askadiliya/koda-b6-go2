# Go Slice & Nested Struct Example

Program ini adalah contoh sederhana penggunaan **slice** dan **nested struct** di bahasa Go (`Golang`). Program ini terdiri dari dua bagian utama: manipulasi slice angka dan penggunaan nested struct untuk menampilkan data.

---

## Fitur

1. **Slice Manipulation**

   * Menampilkan slice awal
   * Menyisipkan elemen baru di tengah slice
   * Menampilkan semua elemen slice
   * Menukar posisi elemen tertentu

2. **Nested Struct Example**

   * Membuat nested struct untuk menyimpan data
   * Menampilkan data dari struct bertingkat
   * Menjumlahkan dua elemen integer dari struct

---


### Bagian 2: Nested Struct Example

Program ini menggunakan beberapa **nested struct** untuk menampilkan informasi seperti:

* "koda"
* "Hello World"
* "Tech Academy"
* "Apple"
* Penjumlahan angka dari dua slice
